======================================
            LOOPS IN BASH
======================================

Loops are used to execute a block of code repeatedly as long as a condition is true.

Bash supports 3 main types of loops:
1Ô∏è‚É£ for loop
2Ô∏è‚É£ while loop
3Ô∏è‚É£ until loop

Also includes control statements:
   break ‚Üí exit loop
   continue ‚Üí skip to next iteration

---------------------------------------
1Ô∏è‚É£ FOR LOOP
---------------------------------------
Used when the number of iterations is known.

Syntax:
for variable in list
do
   commands
done

Example:
for i in 1 2 3 4 5
do
   echo "Number: $i"
done

Using C-style syntax:
for (( i=1; i<=5; i++ ))
do
   echo "i = $i"
done

Iterating through files:
for file in *.txt
do
   echo "Processing $file"
done

---------------------------------------
2Ô∏è‚É£ WHILE LOOP
---------------------------------------
Runs as long as the condition is TRUE.

Syntax:
while [ condition ]
do
   commands
done

Example:
count=1
while [ $count -le 5 ]
do
   echo "Count: $count"
   ((count++))
done

---------------------------------------
3Ô∏è‚É£ UNTIL LOOP
---------------------------------------
Opposite of while ‚Äî runs until condition becomes TRUE.

Syntax:
until [ condition ]
do
   commands
done

Example:
x=1
until [ $x -gt 5 ]
do
   echo "x = $x"
   ((x++))
done

---------------------------------------
4Ô∏è‚É£ NESTED LOOPS
---------------------------------------
One loop inside another.

Example:
for i in 1 2 3
do
   for j in a b c
   do
      echo "$i $j"
   done
done

---------------------------------------
5Ô∏è‚É£ CONTROL STATEMENTS
---------------------------------------
break ‚Üí exits loop completely  
continue ‚Üí skips to next iteration  

Example:
for i in {1..5}
do
  if [ $i -eq 3 ]; then
     continue
  fi
  echo $i
done

---------------------------------------
6Ô∏è‚É£ LOOP + CASE MENU (Practical Use)
---------------------------------------
Used for creating interactive shell menus.

Example:
while true
do
  echo "1. Show Date"
  echo "2. Show Current Dir"
  echo "3. Exit"
  read -p "Enter choice: " c
  case $c in
     1) date ;;
     2) pwd ;;
     3) break ;;
     *) echo "Invalid" ;;
  esac
done
