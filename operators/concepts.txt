==========================
      OPERATORS IN BASH
==========================

Operators are used to perform operations on variables and values in shell scripting.

-------------------------
1Ô∏è‚É£ Arithmetic Operators
-------------------------
Used to perform mathematical calculations.

Syntax:
    $(( expression )) or expr

Common operators:
    +   addition
    -   subtraction
    *   multiplication
    /   division
    %   modulus

Example:
    a=10; b=3
    echo $((a + b))   # 13
    echo $((a % b))   # 1

-------------------------
2Ô∏è‚É£ Relational Operators
-------------------------
Used to compare numbers (not strings).

Syntax:
    [ condition ]

Operators:
    -eq   equal to
    -ne   not equal to
    -gt   greater than
    -lt   less than
    -ge   greater than or equal to
    -le   less than or equal to

Example:
    a=5; b=10
    [ $a -lt $b ] && echo "a is less than b"

-------------------------
3Ô∏è‚É£ Logical Operators
-------------------------
Used to combine multiple conditions.

Operators:
    &&   Logical AND
    ||   Logical OR
    !    Logical NOT

Example:
    [ $a -lt 20 ] && [ $b -gt 5 ] && echo "Both conditions true"

-------------------------
4Ô∏è‚É£ String Operators
-------------------------
Used to compare and check string properties.

Operators:
    =     equal
    !=    not equal
    -z    string is null
    -n    string is not null

Example:
    str="Aakash"
    [ -n "$str" ] && echo "String is not empty"

-------------------------
5Ô∏è‚É£ File Test Operators
-------------------------
Used to test file properties.

Operators:
    -e file   file exists
    -f file   regular file
    -d dir    directory
    -r file   readable
    -w file   writable
    -x file   executable

Example:
    [ -f myfile.txt ] && echo "File exists"
