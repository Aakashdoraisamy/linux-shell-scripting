1. INTRODUCTION
------------------------------------------------------------
Linux is a free, open-source operating system widely used in servers, development, cloud computing, and DevOps. 
The command line (CLI) allows direct interaction with the system using text commands.

Directory structure:
--------------------------------------
/      - Root directory
/home  - User files
/etc   - Configuration files
/var   - Logs and temporary files
/usr   - User programs
/bin   - Essential binaries
/sbin  - System binaries
/tmp   - Temporary files
--------------------------------------

------------------------------------------------------------
2. FILE AND DIRECTORY COMMANDS
------------------------------------------------------------
pwd          - Print current working directory
ls           - List files and directories (ls -l, ls -a)
cd           - Change directory (cd /etc)
mkdir        - Create new directory (mkdir project)
rmdir        - Remove empty directory (rmdir test)
rm           - Remove files/directories (rm -rf temp)
cp           - Copy files (cp file1.txt /home/)
mv           - Move or rename files (mv old new)
touch        - Create empty file (touch test.txt)
cat          - Display file content (cat file.txt)
head         - Show first lines (head -n 5 log.txt)
tail         - Show last lines (tail -f log.txt)
find         - Search files (find /home -name "file.txt")
locate       - Fast search (locate config.yaml)

------------------------------------------------------------
3. FILE PERMISSIONS AND OWNERSHIP
------------------------------------------------------------
chmod        - Change file permissions (chmod 755 script.sh)
chown        - Change file owner (chown user:user file.txt)
chgrp        - Change group (chgrp developers file.txt)
umask        - Default permission mask (umask 022)

Permission Example:
-rwxr-xr--  => Owner (rwx), Group (r-x), Others (r--)

------------------------------------------------------------
4. PROCESS AND SYSTEM MONITORING
------------------------------------------------------------
ps           - Show running processes (ps aux)
top          - Real-time monitor (top)
htop         - Interactive process viewer
kill         - Terminate process (kill 1234)
pkill        - Kill by name (pkill firefox)
nice/renice  - Change process priority (nice -n 10 process)
jobs         - Show background jobs
fg/bg        - Foreground/background job control
uptime       - Show uptime (uptime)

Company Use: Monitor CPU, memory, and process health on servers.

------------------------------------------------------------
5. NETWORKING COMMANDS
------------------------------------------------------------
ping         - Check connectivity (ping google.com)
ifconfig/ip  - Show IP configuration (ip a)
netstat/ss   - Show network connections (ss -tulnp)
curl         - Send HTTP request (curl -I https://example.com)
wget         - Download files (wget https://example.com/file.zip)
scp          - Secure copy (scp file user@ip:/path)
rsync        - Sync files (rsync -avz /src /dest)
ssh          - Remote access (ssh user@192.168.1.5)
traceroute   - Trace route (traceroute google.com)
nslookup     - DNS lookup (nslookup github.com)

------------------------------------------------------------
6. DISK AND STORAGE MANAGEMENT
------------------------------------------------------------
df -h        - Disk usage
du -sh       - Folder size
lsblk        - Block devices
mount        - Mount device (mount /dev/sdb1 /mnt)
umount       - Unmount (umount /mnt)
fdisk -l     - Partition info
blkid        - Device info

------------------------------------------------------------
7. PACKAGE MANAGEMENT
------------------------------------------------------------
For Ubuntu/Debian:
sudo apt update
sudo apt install nginx
sudo apt remove nginx

For CentOS/RHEL:
sudo yum install nginx
sudo yum update
sudo yum remove nginx

Company Use: Install/update software or server tools.

------------------------------------------------------------
8. SEARCH, FILTER, AND TEXT PROCESSING
------------------------------------------------------------
grep         - Search text (grep "error" logfile.log)
awk          - Extract fields (awk '{print $1, $3}' data.txt)
sed          - Replace text (sed 's/error/fixed/g' log.txt)
sort         - Sort lines (sort names.txt)
uniq         - Remove duplicates (uniq sorted.txt)
cut          - Extract columns (cut -d',' -f2 data.csv)
wc           - Count lines/words (wc -l file.txt)

------------------------------------------------------------
9. REDIRECTION AND PIPES
------------------------------------------------------------
>            - Redirect output (echo "Hi" > hello.txt)
>>           - Append output (echo "Hi" >> hello.txt)
<            - Read input (sort < data.txt)
|            - Pipe output (cat file.txt | grep "error")

------------------------------------------------------------
10. COMPRESSION AND ARCHIVING
------------------------------------------------------------
tar          - Archive (tar -cvf backup.tar /home)
gzip         - Compress (gzip file.txt)
gunzip       - Decompress (gunzip file.txt.gz)
zip          - Zip files (zip -r files.zip folder/)
unzip        - Extract (unzip files.zip)

------------------------------------------------------------
11. USER MANAGEMENT
------------------------------------------------------------
whoami       - Show current user
adduser      - Add user (sudo adduser test)
deluser      - Delete user (sudo deluser test)
passwd       - Change password
su           - Switch user (su root)
sudo         - Run as admin (sudo ls /root)

date         - Show date/time

free -h      - Memory usage
uptime       - Uptime info
dmesg        - Kernel messages (dmesg | tail)

14. BONUS TIPS
------------------------------------------------------------

!!                   - Repeat last command
clear / Ctrl+L       - Clear terminal
Ctrl+C               - Stop a running process
Ctrl+D               - Logout from shell
